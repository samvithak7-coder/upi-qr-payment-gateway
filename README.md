# 💳 Secure QR Code Payment Generator

A web application that generates scannable UPI QR codes for digital payments,
enabling direct scan-to-pay via GPay and other UPI apps.

## ✨ Features
- Generates dynamic QR codes with embedded UPI payment data
- Direct scan-to-pay via GPay, PhonePe, Paytm and other UPI apps
- Clean and simple frontend interface
- Backend powered by Flask and Python qrcode library

## 🛠️ Tech Stack
Python · Flask · qrcode · Pillow · HTML · CSS

## 🚀 How to Run
```bash
git clone https://github.com/samvithak7-coder/upi-qr-payment-gateway
cd upi-qr-payment-gateway
pip install flask qrcode pillow
python app.py
```

## 📱 How It Works
1. Enter your UPI ID and payment amount
2. App generates a scannable QR code dynamically
3. Scan with any UPI app to pay instantly
